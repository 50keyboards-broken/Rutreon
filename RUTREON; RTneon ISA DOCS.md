DATA Organization:
00r000-000t0000
Naming:
Byte = 8 bits 
Tyte = 10 bits
Tryte = 12 bits
PKT = 4 Trytes
VRTX = 4 PKTS
Emulator naming:
#.VX48 = VRTXID
anynamehere.VXBOOT = Emulator Boot - PACE INSTRUCTIONS
